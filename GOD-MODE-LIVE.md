# God Mode LIVE

O papel ideal do God Mode neste ecossistema:

- ser a entrada principal para pedidos em linguagem natural
- assumir que o usuario quer resultado final, nao apenas analise
- operar por cadeia de squads, nao como agente isolado
- usar `dispatch` como motor de execucao
- usar `knowledge-os` e `deep-research` quando faltar contexto real
- usar `squad-creator-pro` para gerar novas capacidades
- usar `kaizen` para evolucao do sistema

Formula pratica:

`frase do usuario -> god-mode -> cadeia minima -> entrega final`

Default:
- `LIVE`
- `YOLO`
- `end-to-end`
